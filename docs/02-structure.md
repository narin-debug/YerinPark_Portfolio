# 페이지 구조

원페이지 구성. 아래 순서대로 세로 스크롤하며 섹션이 이어진다.

## 1. 히어로
이름/직함이 텍스트 라인 리빌 효과로 등장. 배경은 대표 편집 작업 이미지
(추후 영상으로 교체 가능).

## 2. 작업 스틸컷 하이라이트 (가로 스크롤 갤러리)
지금까지 편집한 작업물 중 임팩트 있는 스틸컷을 가로로 스크롤하며 보여주는
무드보드 성격의 섹션. 실무 경험뿐 아니라 개인/사이드 작업물도 포함.
캡션은 "다큐 예고편 편집", "인터뷰 리듬 편집"처럼 스킬 중심 문구.
자세한 애니메이션 스펙은 `03-animations.md` 참고.

## 3. 소개 / 스테이트먼트
짧은 자기소개 문단. 텍스트 라인 리빌로 한 줄씩 등장. 실무 경험(한 곳)에서
무엇을 했는지 구체적으로 서술하고, 지향하는 편집 스타일에 대한 관점 포함.

## 4. 프로젝트 그리드 (3열)
포트폴리오의 핵심 섹션. SVG 프레임 카드 + 호버 시 이미지 크로스페이드 +
라임 컬러 스트로크 리빌. 항목 수는 준비된 자료만큼만 구성(개수 강제하지
않음). 클릭 시 실제 영상으로 연결(유튜브/비메오 embed 또는 상세 이동).
자세한 애니메이션 스펙은 `03-animations.md` 참고.

## 5. 툴 / 스킬 밴드
사용 가능한 편집 소프트웨어(Premiere, DaVinci Resolve, After Effects 등)를
얇은 가로 스트립 형태로 나열. 패럴랙스로 스크롤과 함께 살짝 이동.

## 6. 컨택 / 푸터
이메일, 소셜 링크. 마지막 줄은 텍스트 라인 리빌로 마무리.

## 제외한 섹션
- 커리어 하이라이트(경력 나열형): 경력이 한 곳뿐이므로 제외, 대신 2번
  섹션을 "작업 스타일 하이라이트"로 대체
- 클라이언트/방송사 로고 섹션: 제외, 3번(소개) 섹션에 자연스럽게 흡수
